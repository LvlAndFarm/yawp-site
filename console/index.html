Before you start, scaffold and compile a endpoint 

$ mvn yawp:endpoint -Dmodel=person && mvn clean install

Now copy & paste the js below to the YAWP! Sandbox to try out the API 

// create a person
yawp('/people').create({}).done(function(p) { console.log(p); });

// list people
yawp('/people').list(function(p) { console.log(p); });
